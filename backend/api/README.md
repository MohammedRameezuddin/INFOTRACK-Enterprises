# Infotrack API (NestJS skeleton)

This is a minimal NestJS-style skeleton used as a starting point for the Infotrack backend.

Quick start (inside `backend/api`):

```bash
npm install
npm run start
```

The server listens on port `3000` and exposes a sample `GET /products` endpoint.

This is intentionally minimal — replace with full NestJS project structure and persistence (Postgres, TypeORM/Prisma) for production.
